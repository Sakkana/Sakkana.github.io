用户交互Scanner

java.util.Scaaner是jdk5的新特性 通过Scanner类获取用户输入

next()和nextLine()方法获取输入的字符串

再读取前一般要判断是否还有输入的数据

next()

1.一定要读取到有效字符后才可以结束

2.对输入有效字符之前遇到的空白，next()会自动丢弃

3.只有输入有效字符后才将其后面输入的空白作为分隔符/结束符

4.next()不能获取有空格的字符串

import java.util.Scanner;

public class Hello{
    public static void main(String[] args) {
        // 创建一个扫描器对象
        Scanner scanner = new Scanner(System.in);
        System.out.println("Please input:");

        // 判断用户有没有输入
        if(scanner.hasNext()){
            String str = scanner.next();
            System.out.println(str);
        }

        // 关闭io流，否则占用资源
        scanner.close();
    }
}

计算平均值
import java.util.Scanner;

public class Hello{
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        double sum = 0;
        int cnt = 0;

        System.out.println("Please input");
        while (scanner.hasNextDouble()){
            double x = scanner.nextDouble();
            ++ cnt;
            sum += x;
        }

        System.out.println("sum:" + sum);
        System.out.println("average:" + sum / cnt);

        scanner.close();
    }
}

打印三角形

public class Hello{
    public static void main(String[] args) {
        for (int i = 1; i <= 5; ++ i){
            for (int j = 5; j >= i; -- j){
                System.out.print(" ");
            }
            for(int j = 1; j <= 2 * i - 1; ++ j){
                System.out.print("*");
            }
            System.out.println();
        }
    }
}

public class Hello{
    public static void main(String[] args) {
        int cnt = 3;

        outer:for(int i = 1; i <= 5; ++ i){
            System.out.print(i + "\t");
            if(i == 5 && cnt != 0){
                -- cnt;
                i = 0;
                System.out.println();
                continue outer;	// 从outer标记处开始执行
            }
        }

        System.out.println();

    }
}
