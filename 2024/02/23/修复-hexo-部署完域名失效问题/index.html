修复 hexo 部署完域名失效问题

Sakana
2024-02-23
config

问题描述

每次更新完博客并 deploy 之后，访问阿里云域名无法重新解析到 xxx.github.io，一打开就是 404。

解决方案

没有在 hexo 项目中加入 CNAME。

在 hexo 根目录下的 source 目录下新增文件 CNAME，并填入域名。

重新打开网站会发现一切都正常了。
