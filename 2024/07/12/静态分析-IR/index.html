3AC 三地址码

右侧最多只有一个运算符。

可以充当地址的 Address

  • 变量名 x, y, z
  • 常量 3
  • 编译器生成的临时变量 tmp

常用 3AC 形式

  • x = y bop z
  • x = uop y
  • x = y
  • goto L
  • if x goto L
  • if x rop y goto L
