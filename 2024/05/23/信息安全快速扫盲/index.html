    网络安全/系统安全

    病毒

    可以自我复制的程序。

    木马

    表面看起来没有问题，但是含有恶意代码。

    通过木马传播的计算机病毒非常常见。

    rootkit

    一个工具集合，用于掩盖其入侵并提权行为的工具集。

    1. 通过已知漏洞或破解口令 -> 获取用户级访问权限
    2. 安装 rootkit

    白帽黑客攻击 & 渗透测试

    对组织的系统安全性进行测试。

    Shadon

    漏洞搜索引擎。

    黑帽黑客

    使用黑客技术进行非法活动。

    灰帽黑客

    曾经是黑帽子，后来变成了白帽子。

    脚本小子

    随着黑客工具的泛滥，有人下载了工具进行网络工具但是没有了解工具的具体情况。

    红队 & 蓝队

    红队：进行渗透测试，模拟对手的攻击行为。

    蓝队：组织红队进攻的防守队伍。

    端口扫描

    连接目标系统上的每个端口并查看有哪些端口是打开的。

    • ping 扫描
    • 连接扫描
    • SYN 扫描 -> 泛洪DOS攻击
    • FIN 扫描

    漏洞评估

    利用工具对系统评估，检测是否容易收到特定的攻击。

    枚举

    查明目标系统中所有的内容，如一个网络中所有的服务器、计算机、打印机等。

    DOS攻击（拒绝服务攻击）

    不会试图入侵系统而是组织合法用户对系统进行访问。

    缓冲区溢出

    恶意将过量的数据写入缓冲区而导致缓冲区无法容纳。

    SQL 注入

    针对网站攻击最流行的行为。

    向 Web 应用程序传递 SQL 命令，并使得服务器执行。

    XSS 注入（跨站脚本注入）

    将客户端脚本注入用户查看的 Web 页面中（交互区域）。

    用户访问该部分时，执行的是攻击者的脚本，而不是站点预期的功能。

    CSRF（跨站请求伪造）

    Cookie 投毒

    URL 劫持

    无线攻击

    双面恶魔攻击
    WPS 攻击

    口令破解

    防火墙

    网络边界处，根据某些参数（数据包大小、src IP、protocol、port）过滤入站数据包。

    • 内置防火墙的路由器（思科等）
    • 专门运行防火墙软件的专用路由器
